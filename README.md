# ytms
A basic, simple CLI program used to search YTMusic songs, artists, durations and views anonymously and without an account.




I am 12 and currently learning python with Harvard's free CS50p course. I decided to make a search application that runs in the terminal through the CLI. This projects is under the MIT-License. Please fork this and do whatever you want with it.




# How It Works
Either you use CLI Arguments:



#python3 ytms.py "2-word Song"


or


#python3 ytms.py 1-word-song



or you can use the input function inside the program:



#python3 ytms.py

#Pick A Song To Search: very good song!



The program prints out the details in a readable form.



#python3 ytms.py Pizza

#Song = totally real song about pizza

#Artist = totally real artist who loves pizza

#Duration (Minutes) = 1 : 23

#Views = 123M




After, you can either press enter to see the full .json file or press CTRL + C to stop the program.
